# 面向对象程序设计

## 1. 课程学习内容

21级学的C++，C++和Java之间反复横跳+1。这门课属于程序设计课程的进阶版，两门课也是复旦本科阶段仅有的两门会带着学编程语言的课程。相比于上学期的程序设计讲的主要是面向过程的编程理念，这门课主要是讲类的一些概念与编程理念

### 1.1 先修要求

程序设计
